[PROJECT] CS3130, Assign02
          WORD PARSER

          A Python program for generating a frequency distrib
          table for the words in a text file.

          Executing the program displays a menu showing the words
          parsed from the supplied file, its corresponding freqs,
          and a histogram for the frequency distribution.

          At any time, the user may quit the program by pressing
          the [ENTER] or [ESC] keys.
